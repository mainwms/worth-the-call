# Worth the Call?

Worth the Call? is a free senior benefit call checker.

It does not decide eligibility or promise money. It helps users decide which official agency may be worth contacting, what to ask, and what information to have ready.

This tool does not ask for Social Security numbers, Medicare numbers, bank information, full addresses, or private medical records.
